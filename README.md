# AI Entrepreneur Teammate

Discord-native AI teammate for a small founder group.

The product helps the team:

- summarize Discord discussions by session or time window
- research products, markets, competitors, and business ideas
- retain durable memory across decisions and research
- publish useful outputs into GitHub first, then Notion and Google Workspace

## MVP Direction

Build the narrowest useful teammate first:

1. Discord bot with slash commands.
2. Chat summarization for last session and custom time windows.
3. MBA-style research brief generation.
4. Memory system for durable team context.
5. Markdown artifacts in GitHub.
6. Notion and Google Workspace publishing after the core workflow is reliable.
